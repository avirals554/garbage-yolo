1. Project Overview

This project is a real-time garbage and litter detection system. Given a webcam feed, image, or video file, it draws bounding boxes around garbage items and displays a cleanliness score — a percentage showing how clean the scene is.

What it can detect
After fine-tuning: plastic bags, wrappers, cigarette butts, cans, bottles, cardboard, food waste, and 20+ more real-world litter categories from the TACO dataset.

Architecture at a Glance

Input (webcam / image / video) │ ▼ ┌─────────────┐ │ YOLOv8 │ ← fine-tuned on TACO garbage dataset │ (detector) │ detects bounding boxes + class labels └─────────────┘ │ ▼ Filter to garbage classes (all classes if fine-tuned, COCO subset if fallback) │ ▼ Compute cleanliness score (count penalty 60% + coverage penalty 40%) │ ▼ Draw boxes + overlay on frame │ ▼ Display / Save output

2. How YOLOv8 Works

YOLO stands for You Only Look Once. Unlike older detection methods that scan an image multiple times at different scales, YOLO processes the entire image in a single forward pass through a neural network — making it fast enough for real-time video.

The Detection Pipeline

  1. Backbone — A deep convolutional network (CSPDarknet) that extracts features from the image: edges, textures, shapes, and high-level semantic patterns.
  2. Neck — A feature pyramid network (PANet) that combines features at different scales so the model can detect both large and tiny objects in the same image.
  3. Head — Predicts bounding boxes, objectness scores, and class probabilities for each grid cell in the image.

For this project we use Nano (n) as the default — it runs in real-time on a laptop CPU after training. You can switch to Small (s) for noticeably better accuracy if you have a few more minutes of training time.

Transfer Learning

Training a model from scratch requires millions of images and days of compute. Transfer learning shortens this dramatically. We start from yolov8n.pt — weights pre-trained on ImageNet (1.2 million images, 1,000 classes). The model already understands edges, shapes, and textures. We then fine-tune it on our garbage dataset, so it only needs to adapt rather than learn from nothing. This is why 50 epochs on ~1,300 images is enough.

3. Original Code — Explained

The original garbage_detector.py had four main components:

3.1 — GARBAGE_CLASSES Dictionary

A hardcoded mapping of COCO class IDs to display names. These are the classes the pretrained model already knows — things like bottles, cups, and food items used as proxies for garbage. 

# Keys = COCO class IDs, Values = what to show on screen

    41: "cup",
    46: "banana",
    67: "cell phone",  # abandoned phone
    # ... 16 more COCO classes

3.2 — compute_cleanliness()

Takes the number of detected garbage items and their bounding boxes, returns a cleanliness score from 0.0 to 1.0 and a status label. 

def compute_cleanliness(garbage_count, frame_area, boxes):
    if garbage_count == 0:
        return "CLEAN", 1.0

    # How much of the frame is covered by garbage bounding boxes
    coverage = sum((x2-x1)*(y2-y1) for x1,y1,x2,y2 in boxes)
    coverage_ratio = min(coverage / frame_area, 1.0)

    count_penalty = min(garbage_count / 10.0, 1.0)

    # 60% weight on item count, 40% on screen coverage
    score = max(0.0, 1.0 - (0.6 * count_penalty + 0.4 * coverage_ratio))

    if   score >= 0.7: status = "MOSTLY CLEAN"
    elif score >= 0.4: status = "DIRTY"
    else:             status = "VERY DIRTY"

    return status, score
Score Formula
score = 1.0 − (0.6 × count_penalty + 0.4 × coverage_ratio)

Count penalty: number of items ÷ 10, capped at 1.0
Coverage ratio: total box area ÷ frame area, capped at 1.0
Count is weighted more (60%) because a dense pile of small items is worse than one large object.

3.3 — draw_garbage_box()

Draws a red rectangle around each detected item and puts the class name + confidence % above it.

def draw_garbage_box(frame, x1, y1, x2, y2, label, conf):
    cv2.rectangle(frame, (x1,y1), (x2,y2), (0,0,255), 2)
    text = f"{label} {conf:.0%}"
    (tw, th), _ = cv2.getTextSize(text, FONT, 0.55, 1)
    # Solid red background behind the text so it's readable over any image
    cv2.rectangle(frame, (x1, y1-th-8), (x1+tw+6, y1), (0,0,255), -1)
    cv2.putText(frame, text, (x1+3, y1-4), FONT, 0.55, (255,255,255), 1, cv2.LINE_AA)

3.4 — process_frame() and run()

process_frame() runs YOLO on one frame, filters results to only garbage classes, draws boxes, and computes the score. run() is the main loop that opens the video source and calls process_frame() on every frame.

4. The Problem with the Original Approach

Core Limitation
The original code uses a model trained on generic objects (COCO dataset) and pretends that certain objects are garbage. It never actually learned what litter looks like.

What it misses entirely:

  • Plastic bags, cling wrap, shopping bags
  • Cigarette butts
  • Candy wrappers, crisp packets
  • Cardboard scraps, newspaper
  • Styrofoam cups/containers
  • Any garbage that is crumpled, crushed, or partially obscured
  • Mixed piles of unidentifiable rubbish

What it falsely detects:

  • A banana in a fruit bowl (labelled as litter)
  • A book on a shelf (labelled as litter)
  • A cup of coffee on your desk (labelled as litter)

The fix is to train on real, labelled photos of actual litter in real environments — which is exactly what the TACO dataset provides.

5. The Solution — Fine-Tuning on Real Garbage Data

TACO (Trash Annotations in Context) is an open-source dataset of 1,500+ photos of real-world litter — on beaches, streets, parks, and indoors — with precise bounding box annotations across 28 garbage categories.

6. File: download_dataset.py

This script downloads the TACO dataset and converts it into the format YOLOv8 expects. You don't need to run this manually if you use the Kaggle notebook — it's embedded there.

What it does step by step:

  1. Downloads annotations.json from TACO's GitHub (COCO format)
  2. Downloads TACO's own image downloader script
  3. Runs the image downloader to fetch ~1,500 photos from Flickr
  4. Converts COCO bounding boxes to YOLO format
  5. Splits images into train (85%) and validation (15%) sets
  6. Writes dataset/data.yaml — the config file YOLOv8 reads

Key function — COCO to YOLO conversion:

Coordinate Format Difference
COCO format: [x_top_left, y_top_left, width, height] in absolute pixels
YOLO format: [center_x, center_y, width, height] normalised 0–1 relative to image size 
def coco_bbox_to_yolo(bbox, img_w, img_h):
    x, y, w, h = bbox
    # Shift from top-left corner to centre point
    cx = (x + w / 2) / img_w
    cy = (y + h / 2) / img_h
    nw = w / img_w
    nh = h / img_h
    return cx, cy, nw, nh

Category mapping:

TACO has 60 fine-grained categories (e.g. "Clear plastic bottle", "Drink can", "Crisp packet"). We collapse these into their supercategories (e.g. "Bottle", "Can", "Wrapper") to reduce class count and avoid having too few examples per class. The SINGLE_CLASS toggle at the top of the file collapses everything to one class called "garbage" if you want the simplest possible model.

Full source code:

"""
Download and prepare the TACO dataset for YOLOv8 fine-tuning.
"""

import json, os, shutil, urllib.request
from pathlib import Path
from collections import defaultdict
import random

ANNOTATIONS_URL = "https://raw.githubusercontent.com/pedropro/TACO/master/data/annotations.json"
TACO_DOWNLOADER_URL = "https://raw.githubusercontent.com/pedropro/TACO/master/download.py"

SINGLE_CLASS = False   # True = one "garbage" class; False = 28 categories
DATASET_DIR  = Path("dataset")
VAL_SPLIT    = 0.15


def download_file(url, dest):
    print(f"  Downloading {url.split('/')[-1]} ...")
    urllib.request.urlretrieve(url, dest)


def coco_bbox_to_yolo(bbox, img_w, img_h):
    x, y, w, h = bbox
    return (x+w/2)/img_w, (y+h/2)/img_h, w/img_w, h/img_h


def build_dataset(annotations_path):
    with open(annotations_path) as f:
        coco = json.load(f)

    categories = {cat["id"]: cat["name"] for cat in coco["categories"]}
    supercats  = {cat["id"]: cat.get("supercategory", cat["name"]) for cat in coco["categories"]}

    if SINGLE_CLASS:
        class_names  = ["garbage"]
        cat_to_class = {cid: 0 for cid in categories}
    else:
        unique_supers  = sorted(set(supercats.values()))
        super_to_idx   = {s: i for i, s in enumerate(unique_supers)}
        class_names    = unique_supers
        cat_to_class   = {cid: super_to_idx[supercats[cid]] for cid in categories}

    img_annotations = defaultdict(list)
    for ann in coco["annotations"]:
        img_annotations[ann["image_id"]].append(ann)

    images = coco["images"]
    random.shuffle(images)
    n_val   = max(1, int(len(images) * VAL_SPLIT))
    val_ids = {img["id"] for img in images[:n_val]}

    for split in ("train", "val"):
        (DATASET_DIR / "images" / split).mkdir(parents=True, exist_ok=True)
        (DATASET_DIR / "labels" / split).mkdir(parents=True, exist_ok=True)

    written, skipped = {"train": 0, "val": 0}, 0

    for img_info in images:
        src = DATASET_DIR / "raw" / img_info.get("file_name", "")
        if not src.exists(): skipped += 1; continue

        split = "val" if img_info["id"] in val_ids else "train"
        shutil.copy2(src, DATASET_DIR / "images" / split / src.name)

        lines = []
        for ann in img_annotations[img_info["id"]]:
            cx, cy, nw, nh = coco_bbox_to_yolo(
                ann["bbox"], img_info["width"], img_info["height"])
            if nw > 0 and nh > 0:
                lines.append(f"{cat_to_class[ann['category_id']]} {cx:.6f} {cy:.6f} {nw:.6f} {nh:.6f}")
        if not lines: continue

        (DATASET_DIR/"labels"/split/(src.stem+".txt")).write_text("\n".join(lines))
        written[split] += 1

    print(f"Written: train={written['train']}, val={written['val']}, skipped={skipped}")
    return class_names


def write_yaml(class_names):
    yaml_content = f"""path: {DATASET_DIR.resolve()}
train: images/train
val: images/val
nc: {len(class_names)}
names: {class_names}
"""
    yaml_path = DATASET_DIR / "data.yaml"
    yaml_path.write_text(yaml_content)
    return yaml_path


def main():
    DATASET_DIR.mkdir(exist_ok=True)
    ann_path = DATASET_DIR / "annotations.json"
    if not ann_path.exists():
        download_file(ANNOTATIONS_URL, ann_path)

    raw_dir  = DATASET_DIR / "raw"
    raw_dir.mkdir(exist_ok=True)
    taco_dl  = DATASET_DIR / "taco_download.py"
    if not taco_dl.exists():
        download_file(TACO_DOWNLOADER_URL, taco_dl)

    os.system(f"python {taco_dl} --dataset_path {raw_dir} --ann_file {ann_path}")

    class_names = build_dataset(ann_path)
    write_yaml(class_names)
    print("Done! Run train.py next.")

if __name__ == "__main__":
    main()

7. File: train.py

This is the fine-tuning script. It loads the pre-trained YOLOv8 weights, trains on the TACO dataset, and saves the best checkpoint.

Augmentation Settings — Why These Specific Values

Data augmentation artificially multiplies your dataset by creating variations of each image. For garbage detection specifically:

ParameterValueWhy
hsv_h=0.02±2% hueLitter fades and discolours in sunlight
hsv_s=0.6±60% saturationOvercast vs sunny conditions look very different
hsv_v=0.4±40% brightnessIndoor fluorescent vs outdoor daylight
fliplr=0.550% chanceLitter has no preferred orientation
flipud=0.110% chanceOccasional overhead/tilted camera angles
mosaic=1.0Always onMixes 4 images; teaches model to spot small items
scale=0.5±50% zoomGarbage at different distances
degrees=10.0±10° rotationCamera tilt and uneven ground

Early Stopping (patience=20)

If validation mAP doesn't improve for 20 consecutive epochs, training stops automatically. This prevents overfitting and saves time — a model that's peaked at epoch 35 doesn't benefit from running to epoch 50.

Full source code:

"""
Fine-tune YOLOv8 on the TACO garbage dataset.
Prerequisites: python download_dataset.py
"""

import argparse
from pathlib import Path
from ultralytics import YOLO

DATASET_YAML  = Path("dataset/data.yaml")
DEFAULT_MODEL = "n"
DEFAULT_EPOCHS = 50
DEFAULT_BATCH  = 16
RUN_NAME       = "garbage_finetune"


def train(model_size, epochs, batch, imgsz, resume, device):
    if not DATASET_YAML.exists():
        print("ERROR: Run 'python download_dataset.py' first.")
        return

    if resume:
        last_ckpt = Path(f"runs/detect/{RUN_NAME}/weights/last.pt")
        if last_ckpt.exists():
            model = YOLO(str(last_ckpt))
            model.train(resume=True)
            return

    model = YOLO(f"yolov8{model_size}.pt")
    model.train(
        data     = str(DATASET_YAML),
        epochs   = epochs,
        imgsz    = imgsz,
        batch    = batch,
        name     = RUN_NAME,
        device   = device,

        # Augmentation
        hsv_h    = 0.02,   hsv_s = 0.6,   hsv_v     = 0.4,
        fliplr   = 0.5,   flipud = 0.1,  mosaic    = 1.0,
        scale    = 0.5,   translate = 0.1, degrees  = 10.0,

        patience    = 20,
        save_period = 10,
        plots       = True,
        val         = True,
    )
    print(f"Best weights: runs/detect/{RUN_NAME}/weights/best.pt")


def validate(weights_path):
    model   = YOLO(weights_path)
    results = model.val(data=str(DATASET_YAML), imgsz=640)
    print(f"mAP50: {results.box.map50:.3f} | Precision: {results.box.mp:.3f} | Recall: {results.box.mr:.3f}")


def main():
    parser = argparse.ArgumentParser()
    parser.add_argument("--model",    default="n", choices=["n","s","m","l","x"])
    parser.add_argument("--epochs",   type=int,   default=50)
    parser.add_argument("--batch",    type=int,   default=16)
    parser.add_argument("--imgsz",   type=int,   default=640)
    parser.add_argument("--device",  default="")
    parser.add_argument("--resume",  action="store_true")
    parser.add_argument("--validate", metavar="WEIGHTS")
    args = parser.parse_args()

    if args.validate:
        validate(args.validate)
    else:
        train(args.model, args.epochs, args.batch, args.imgsz, args.resume, args.device)

if __name__ == "__main__":
    main()

8. File: garbage_detector.py (Updated)

The detector was updated to support two modes: a fine-tuned model (recommended) and the original COCO fallback. The mode is selected automatically based on whether you pass the --weights flag.

The Two-Mode Design in process_frame()

def process_frame(model, frame, conf_threshold, class_names):
    """
    class_names:
      None → fine-tuned mode: every detected class is garbage
      dict → COCO fallback: filter to class IDs in dict
    """
    results = model(frame, conf=conf_threshold, verbose=False)[0]
    garbage_boxes, garbage_count = [], 0

    for box in results.boxes:
        cls_id = int(box.cls[0])
        conf   = float(box.conf[0])
        x1, y1, x2, y2 = map(int, box.xyxy[0])

        if class_names is None:
            # Fine-tuned: model's own class names, all are garbage
            label = results.names[cls_id]
        else:
            # COCO fallback: skip anything not in our proxy list
            if cls_id not in class_names: continue
            label = class_names[cls_id]

        draw_garbage_box(frame, x1, y1, x2, y2, label, conf)
        garbage_boxes.append((x1, y1, x2, y2))
        garbage_count += 1

    status, score = compute_cleanliness(garbage_count, h*w, garbage_boxes)
    return frame, status, score, garbage_count

Full source code:

import argparse, cv2
from pathlib import Path
from ultralytics import YOLO

COCO_GARBAGE_CLASSES = {
    39: "bottle", 40: "wine glass", 41: "cup", 42: "fork",
    43: "knife",  44: "spoon",      45: "bowl", 46: "banana",
    47: "apple",  48: "sandwich",   49: "orange", 50: "broccoli",
    54: "donut",  67: "cell phone", 73: "book",
}

GARBAGE_COLOR = (0, 0, 255)
CLEAN_COLOR   = (0, 200, 0)
LABEL_BG      = (20, 20, 20)
FONT          = cv2.FONT_HERSHEY_SIMPLEX


def compute_cleanliness(garbage_count, frame_area, boxes):
    if garbage_count == 0: return "CLEAN", 1.0
    coverage       = sum((x2-x1)*(y2-y1) for x1,y1,x2,y2 in boxes)
    coverage_ratio = min(coverage / frame_area, 1.0)
    count_penalty  = min(garbage_count / 10.0, 1.0)
    score = max(0.0, 1.0 - (0.6 * count_penalty + 0.4 * coverage_ratio))
    status = "MOSTLY CLEAN" if score >= 0.7 else "DIRTY" if score >= 0.4 else "VERY DIRTY"
    return status, score


def draw_garbage_box(frame, x1, y1, x2, y2, label, conf):
    cv2.rectangle(frame, (x1,y1), (x2,y2), GARBAGE_COLOR, 2)
    text = f"{label} {conf:.0%}"
    (tw,th),_ = cv2.getTextSize(text, FONT, 0.55, 1)
    cv2.rectangle(frame, (x1,y1-th-8), (x1+tw+6,y1), GARBAGE_COLOR, -1)
    cv2.putText(frame, text, (x1+3,y1-4), FONT, 0.55, (255,255,255), 1, cv2.LINE_AA)


def draw_status_overlay(frame, status, score, count, model_tag):
    h, w = frame.shape[:2]
    overlay = frame.copy()
    cv2.rectangle(overlay, (0,0), (w,50), LABEL_BG, -1)
    cv2.addWeighted(overlay, 0.7, frame, 0.3, 0, frame)
    color = CLEAN_COLOR if status=="CLEAN" else (0,200,255) if status=="MOSTLY CLEAN" else GARBAGE_COLOR
    cv2.putText(frame, f"[{model_tag}]  {status}  |  Items: {count}  |  Score: {score:.0%}",
               (10,33), FONT, 0.68, color, 2, cv2.LINE_AA)
    bar_w = int(w * score)
    cv2.rectangle(frame, (0,55), (w,61), (50,50,50), -1)
    cv2.rectangle(frame, (0,55), (bar_w,61), color, -1)


def process_frame(model, frame, conf_threshold, class_names):
    h, w = frame.shape[:2]
    results = model(frame, conf=conf_threshold, verbose=False)[0]
    garbage_boxes, garbage_count = [], 0
    for box in results.boxes:
        cls_id = int(box.cls[0])
        conf   = float(box.conf[0])
        x1,y1,x2,y2 = map(int, box.xyxy[0])
        if class_names is None:
            label = results.names[cls_id]
        else:
            if cls_id not in class_names: continue
            label = class_names[cls_id]
        draw_garbage_box(frame, x1,y1,x2,y2, label, conf)
        garbage_boxes.append((x1,y1,x2,y2))
        garbage_count += 1
    status, score = compute_cleanliness(garbage_count, h*w, garbage_boxes)
    return frame, status, score, garbage_count


def run(source, conf_threshold, save, model_size, weights_path):
    if weights_path:
        model       = YOLO(weights_path)
        class_names = None
        model_tag   = "FINETUNED"
    else:
        model       = YOLO(f"yolov8{model_size}.pt")
        class_names = COCO_GARBAGE_CLASSES
        model_tag   = "COCO-fallback"

    is_webcam  = str(source).isdigit()
    source_val = int(source) if is_webcam else source
    cap = cv2.VideoCapture(source_val)

    writer   = None
    is_image = not is_webcam and Path(str(source)).suffix.lower() in {".jpg",".jpeg",".png"}

    while True:
        ret, frame = cap.read()
        if not ret: break
        processed, status, score, count = process_frame(model, frame, conf_threshold, class_names)
        draw_status_overlay(processed, status, score, count, model_tag)
        cv2.imshow("Garbage Detector", processed)
        if is_image:
            cv2.waitKey(0); break
        if cv2.waitKey(1) & 0xFF == ord("q"): break

    cap.release()
    cv2.destroyAllWindows()


def main():
    parser = argparse.ArgumentParser()
    parser.add_argument("--source",  default="0")
    parser.add_argument("--conf",    type=float, default=0.35)
    parser.add_argument("--save",    action="store_true")
    parser.add_argument("--model",   default="n", choices=["n","s","m","l","x"])
    parser.add_argument("--weights", help="Path to fine-tuned .pt file")
    args = parser.parse_args()
    run(args.source, args.conf, args.save, args.model, args.weights)

if __name__ == "__main__":
    main()

Precision vs Recall tradeoff
For a garbage detector, Recall is more important than Precision. It's better to occasionally flag a clean scene as dirty (false alarm) than to miss actual garbage (false negative). You can tune this by lowering --conf (e.g. 0.25) to catch more garbage at the cost of more false alarms.
